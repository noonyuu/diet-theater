# 国会劇場

## 制作背景

「長くて難しい国会に終止符を！！」
国を知ろうとしてたとえばyoutubeにあがってる国会中継を見ても長くて内容も難しくほとんどの人が挫折してしまうのではないかと考え、とにかくわかりやすく見て、せっかく興味を持ちかけても挫折してしまうというような人を少しでもなくそうというコンセプトです

## 作品概要

国立国会図書館が公開している国会議事録APIを使用してそれらをOpenAIで要約してアニメ帳で劇をみているかのように表示する
